## 4. 主要内容区 (Content)

- **外边距**：8px
- **背景**：白色圆角卡片，`borderRadius: 8`
- **布局**：`flex` 纵向，填充剩余空间

内容区根据当前选中的菜单项渲染不同的 Tab 组件（`OverviewTab`、`UsersTab`、`OrdersTab`、`SmsTab`、`ParcelsTab`、`AdminsTab`、`RolesTab`）。其中 `系统管理员` 菜单进入后，在内容区顶部使用 Tabs 在 `管理员` 与 `系统管理权限` 两个子页面之间切换。

### 4.1 总览页 (OverviewTab)

独立统计卡片布局，不使用以下三层结构。

### 4.2 业务 Tab 通用三层结构

所有业务 Tab（会员、订单、简讯、包裹、物流商、库位、地址簿、单号库等）共享以下纵向三层布局：

```
┌─────────────────────────────────────────────────────────────┐
│                  顶层 - 操作栏（不收缩）                     │
│  ┌────────────────────────────────┐  ┌──────┐               │
│  │  🔍 搜索输入框（含 allowClear）  │  │ 搜索 │   [新增]     │
│  └────────────────────────────────┘  └──────┘               │
├─────────────────────────────────────────────────────────────┤
│                  中层 - 数据表格（填充）                     │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ 表头行 1：列标题（支持排序点击）                        │    │
│  │ 表头行 2：列级搜索输入框 / 日期范围选择器               │    │
│  ├─────────────────────────────────────────────────────┤    │
│  │                    数据行                            │    │
│  │                    数据行                            │    │
│  │                    ...                              │    │
│  │            （虚拟高度自适应）                         │    │
│  └─────────────────────────────────────────────────────┘    │
├─────────────────────────────────────────────────────────────┤
│                  底层 - 分页器（不收缩）                     │
│              ◀ 1 2 3 ... ▶  每页 50 条  跳至 __             │
└─────────────────────────────────────────────────────────────┘
```

#### 顶层 - 操作栏

| 元素 | 说明 |
|---|---|
| 搜索输入框 | 全局关键词搜索，支持 allowClear（点击 X 自动触发重置） |
| 搜索按钮 | 触发全局搜索请求 |
| 新增按钮 | 打开新增/创建弹窗 |
| 其他功能 | 导出、导入、批量删除、物流商过滤（根据 Tab 类型）|

- **样式**：`padding: 12px 16px`，底部 `1px solid #f0f0f0` 分割线
- **布局**：`<Space wrap>`，不可收缩（`flexShrink: 0`）
- **响应式**：按钮排列支持换行（Space wrap）

#### 中层 - 数据表格

| 特性 | 说明 |
|---|---|
| 组件 | Ant Design `<Table>` |
| 双行表头 | 通过 `columns.children` 实现。父列：标题 + 排序；子列：列级搜索输入框 |
| 固定列 | 左侧固定：序号列（含行复选框）；右侧固定：操作列 |
| 列级搜索 | 文本列：`<Input>` 回车触发；日期列：`<DatePicker.RangePicker>` 支持日期范围 |
| 排序 | 点击父表头触发，服务端排序 |
| 虚拟高度 | 通过 `ResizeObserver` 动态计算 `scroll.y`，自适应容器高度 |
| 弹性空白列 | `spacer` 列自动填充剩余空间，避免列堆积 |
| 行操作 | 编辑（打开编辑弹窗）、删除（带确认）、其他（查看详情、导出等） |

- **布局**：`flex: 1, minHeight: 0, overflow: hidden`（重要）
- **scroll.x**：按叶子列 width 总和生成数值型，避免用 `max-content`

#### 底层 - 分页器

| 元素 | 说明 |
|---|---|
| 组件 | 自定义 `<Pagination>` / Ant Design Pagination |
| 功能 | 页码切换、每页条数切换（10/20/30/50）、快速跳转、总条数/页数显示 |

- **样式**：`padding: 6px 16px`，顶部分割线，右对齐
- **不收缩**：`flexShrink: 0`，`z-index: 10`
- **显示信息**：总条数、当前页码、每页条数

### 4.2.1 详情页窗口（弹窗）通用容器布局（detail-modal）

当前各业务模块的详情/编辑/创建窗口统一使用 `detail-modal` 容器框架，结构如下：

```
┌─────────────────────────────────────────────────────────────┐
│ Modal Root（遮罩层 + 居中）                                  │
│  ┌───────────────────────────────────────────────────────┐  │
│  │ .ant-modal（面板定位层）                                │  │
│  │  ┌─────────────────────────────────────────────────┐   │  │
│  │  │ .ant-modal-container                          │   │  │
│  │  │  背景：#f5f5f5（浅灰），padding: 0             │   │  │
│  │  ├─────────────────────────────────────────────────┤   │  │
│  │  │ .ant-modal-header（顶部标题容器）               │   │  │
│  │  │  背景：#f5f5f5，标题/关闭按钮深色               │   │  │
│  │  ├─────────────────────────────────────────────────┤   │  │
│  │  │ .ant-modal-body（中间内容容器）                 │   │  │
│  │  │  背景：#ffffff（白色）                           │   │  │
│  │  │  ┌───────────────────────────────────────────┐  │   │  │
│  │  │  │ Form / Descriptions / Table 等业务内容   │  │   │  │
│  │  │  └───────────────────────────────────────────┘  │   │  │
│  │  ├─────────────────────────────────────────────────┤   │  │
│  │  │ .ant-modal-footer（底部按钮容器）               │   │  │
│  │  │  背景：#f5f5f5，按钮区固定在底部                │   │  │
│  │  └─────────────────────────────────────────────────┘   │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

| 容器层级 | 样式与行为 |
|---|---|
| `.ant-modal-container` | 统一浅灰外层背景，去除默认外边距留白 |
| `.ant-modal-header` | 作为标题栏容器，统一浅灰背景、分割线 |
| `.ant-modal-body` | 作为详情内容主容器，统一白底（表单/详情区域） |
| `.ant-modal-footer` | 作为按钮栏容器，统一浅灰背景、分割线 |

> 说明：提(运)单详情页属于长表单模式，仍使用同一容器框架；仅在业务内容区内部增加分段和滚动控制。

### 4.3 系统管理员页结构

`系统管理员` 作为一级菜单进入后，内容区采用“顶部 Tabs + 子页面内容”的结构：

```
┌─────────────────────────────────────────────────────────────┐
│ Tabs： [平台管理员] [物流商管理员]                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  当前 Tab 对应内容                                           │
│  - 平台管理员：平台级管理员列表（平台账号可见）                 │
│  - 物流商管理员：物流商级管理员列表                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 区域 | 说明 |
|---|---|
| 顶部 Tabs | 位于内容区顶部，底部带 `1px solid #f0f0f0` 分割线；用于切换 `平台管理员` 与 `物流商管理员` |
| 子页面内容 | Tabs 下方为具体页面内容区域，保持 `flex: 1`、`minHeight: 0`，复用原 `AdminsTab` 组件 |

## 6. 关键技术规范

### 表格布局与分页

- **三层结构**：操作栏（flexShrink: 0） + 表格区（flex: 1） + 分页栏（flexShrink: 0）
- **父级 flex 填充**：嵌套 Tabs 中的 Tab 组件要用 `flex: 1, minHeight: 0` 避免溢出
- **虚拟滚动**：表格高度由 `ResizeObserver` 动态计算，自适应容器
- **弹性空白列**：`spacer` 列自动填充剩余宽度
- **scroll.x 设置**：按叶子列宽总和生成数值，勿用 `max-content`

### 表单与弹窗

- **长表单弹窗**：Modal body 加 `max-height: calc(100vh - 200px); overflow-y: auto`
- **表单分区**：按业务逻辑分组（基础信息、参与方、关联数据、备注等）
- **字段校验**：后端参数化验证，前端 Form 布局一致

### 权限与账号隔离

- **平台 vs 物流商**：通过 `role_scope` 区分，数据按 `logistics_provider_id` 做 scope 隔离
- **权限矩阵**：RolesTab 定义权限组，AdminsTab 选角色时显示其权限
- **自动拼接**：物流商管理员账号自动拼 `@代号`
- **字段预置**：编辑模式预置当前值，防止跨账号操作

### 国际化与本地化

- **四语支持**：简体中文、繁体中文、English、Bahasa Indonesia
- **地区推导**：由省名推导 region（CN/TW/HK/MO）和电话区号
- **菜单/标签**：i18n 文件中定义各项菜单项、按钮、提示文字

## 7. UI/UX 设计细节

### 配色方案
- **主色**：橙色 `#f58220`（菜单选中、操作按钮、链接）
- **辅助**：灰色（禁用、次要文字）
- **警示**：红色（删除、异常）
- **成功**：绿色（完成、保存）

### 间距与对齐
- **操作栏内边距**：`padding: 12px 16px`
- **卡片内边距**：由 Card 的 bodyStyle 设置
- **分页栏内边距**：`padding: 6px 16px`
- **菜单项高度**：34px，无圆角

### 响应式与适配
- **侧边栏宽度变化**：展开 160px ↔ 折叠 52px（非 56px）
- **固定列**：左侧序号+复选框、右侧操作列固定
- **日期选择**：DatePicker showTime，支持时间段筛选
- **下拉菜单**：异步搜索列表（如会员选择）支持 debounce
